# Kept Point Backend

## Frameworks and Tools

- **Runtime**: Node.js  
- **Web Framework**: Fastify  
- **Database**: PostgreSQL (with Prisma)  
- **Authentication**: JSON Web Token (JWT)  
- **API Documentation**: Scalar
